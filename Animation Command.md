# Animated Neon Command v1
```
figlet -f slant "NEON LIGHT" | lolcat -a -d 10 -s 30
```

# Animated Neon Command v2
```
# This will animate for a short moment then proceed to the menu
figlet -f slant "LOADING..." | lolcat -a -d 20
clear
```

# Glitch Style
```
figlet -f block "GLITCH" | lolcat -a -s 100 -p 0.5
```

# Vertical Rainbow
```
figlet -f shadow "MATRIX" | lolcat -a -F 0.1
```
