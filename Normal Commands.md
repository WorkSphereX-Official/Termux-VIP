# The Slant Style
```
figlet -f slant "YOUR TEXT" | lolcat
```

# Bold Style 
```
figlet -f block "BOLD" | lolcat
```

# Shadow style (looks 3D)
```
figlet -f shadow "SECURE" | lolcat
```

# Rainbow Effect
Fast Rainbow:
```
figlet "TEXT" | lolcat -p 1
```
Slow/Smooth Rainbow:
```
figlet "TEXT" | lolcat -p 10
```
